# Glossary

| Term | Definition |
|------|------------|
| **Time Capsule** | The Convention Time Capsule product — a durable archive of lessons from the 30th AFL-CIO Convention |
| **Submission** | A lesson-learned record contributed by a submitter (domain, body, contact, optional attachments) |
| **Classification** | Team-lead metadata applied after submit (tags, severity, area, follow-up status, cross-links) |
| **Lookup table** | Canonical categories, tags, and synonyms used to normalize and suggest values during submission |
| **Active submission mode** | GitHub Pages app state where authenticated staff compose and submit lessons |
| **Team-lead mode** | Elevated app state for post-submit classification and enrichment |
| **HTML archive** | Self-contained offline export of the corpus (time-capsule generation) |
| **Convention (30th)** | 30th AFL-CIO Constitutional Convention — June 7–10, 2026, Minneapolis; single-archive scope for v1 |
| **Knowledge domain** | Subject area for a submission: technical, logistical, attendee satisfaction, vendor, morale, or health/safety |
| **gws** | [Google Workspace CLI](https://github.com/googleworkspace/cli) — maintainer command-line tool for Sheets and Drive (`scripts/google/`) |
| **Milestone (M-###)** | Bite-sized developer assignment on ROADMAP.md; objectively verifiable |
| **Stage marker** | ROADMAP `# ── Stage ──` divider; milestones belong to the stage above |
| **Deep Backlog** | Epics in manifest/Deep-Backlog.md; promoted to ROADMAP as M-### when scoped |
| **Number One** | Nickname for the primary coding agent in this repo |
| **make it so** | Execution trigger: build the active plan in agent mode |
