# Release v0.1.0

First public release of the mka-bootstrap governance template.

## Milestones closed

| Milestone | Summary |
|-----------|---------|
| M-001 | Documentation taxonomy, manifest skeleton, architecture and approach docs |
| M-002 | Four governance skills, platform stubs, AGENTS.md, Cursor rules |
| M-003 | init, validate, bootstrap, install scripts; offline test harness |
| M-004 | publish-to-install.sh with install branch overlays |

## Deliverables

- Stack-agnostic documentation architecture (manifest, docs/developer, user, admin)
- Cross-platform agent skills in `docs/developer/skills/` with routing stubs
- `main` / `install` branch model with `publish-to-install.sh`
- Offline `.local-test/` harness with local bare remotes
- Agent invocation conventions (Number One, make it so)
